## Beitrag leisten

Wenn du einen neuen Use-Case umgesetzt hast, ergänze ihn bitte hier:

- Idee kurz beschreiben
- Welche Source-States werden benötigt?
- Welche Items müssen im Adapter angelegt werden?
- Welche Formel wird genutzt?
- Typische Stolpersteine / Checks (z.B. Vorzeichen, Einheiten)
